Reports String concatenations that can be simplified by replacing them with the string template.

Requirements:

  • Concatenation involves multiple variables or literals.
  • Concatenation is not yet in template form.

Example:


  String name = "world";
  long age = 13_750_000_000L;
  String greeting = "Hello, " + name + ". You are " + age + " years old.";

After the quick-fix is applied:


  String name = "world";
  long age = 13_750_000_000L;
  String greeting = STR."Hello, \{name}. You are \{age} years old.";

This inspection only reports if the language level of the project or module is 21 or higher.
